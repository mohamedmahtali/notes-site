---

## Définition

Amazon S3 (Simple Storage Service) est le service d'object storage d'AWS. Il stocke des objets (fichiers) dans des buckets avec une durabilité de 99.999999999% (11 neuf). Utilisé pour les backups, les assets statiques, les artefacts de build, les logs.

---

## Commandes essentielles

```bash
# Créer un bucket
aws s3 mb s3://my-unique-bucket-name --region eu-west-1

# Upload
aws s3 cp ./file.txt s3://my-bucket/
aws s3 sync ./dist/ s3://my-bucket/static/    # sync répertoire

# Download
aws s3 cp s3://my-bucket/file.txt ./
aws s3 sync s3://my-bucket/backups/ ./backups/

# Lister
aws s3 ls s3://my-bucket/
aws s3 ls s3://my-bucket/ --recursive --human-readable

# Supprimer
aws s3 rm s3://my-bucket/file.txt
aws s3 rm s3://my-bucket/ --recursive
```

## Classes de stockage

| Classe | Latence | Coût | Usage |
|---|---|---|---|
| Standard | ms | $$$ | Accès fréquent |
| Standard-IA | ms | $$ | Accès rare |
| Glacier Instant | ms | $ | Archives avec accès rapide |
| Glacier Flexible | minutes | $$ | Archives longue durée |
| Deep Archive | heures | $ | Archives très longue durée |
